road-bikelanes
=========
Supports ArcGIS versions: 10.3.1, 10.4

Script to add bike lane attributes to Statewide Streets.

### Running the script
- Set the Workspaces for temp data and outputs.
- The script will produce a table named LineCoverage that can be used to populate bike_l, bike_r and bike_status fields in StatewideStreets.
